# Organic SHA-256
TypeScriptで0から書いたSHA-256のアルゴリズムです。全角文字、絵文字、アラビア語、ロシア語でテスト済みです。

n番煎じなのでパッケージ化はしませんが気が向いたらもっと速くできないか考えます。めっちゃ速くなったらパッケージ化します。

# Development
```
yarn install
```

PR前

```
yarn test
yarn format
```

huskyとかはコミット多くなったら導入します。
